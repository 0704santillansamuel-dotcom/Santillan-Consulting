# Santillan-Consulting
Desarrollo de una landing page corporativa para Santillán Consulting, orientada a la presentación institucional de servicios de consultoría estratégica.

El proyecto fue construido con HTML, CSS y JavaScript dentro de un enfoque frontend simple pero profesional, priorizando experiencia de usuario, diseño responsive y comunicación empresarial.

Puedes ver la versión desplegada del sitio en:
https://santillanconsulting.netlify.app
